# **在线考试系统**

## **1、系统角色与核心功能**

### **1.1、管理员端**

系统运维与基础数据管理，管理员是整个系统的运维核心，负责基础数据配置和账号管理。

| 功能模块      | 具体功能     | 设计说明                                       |
| :------------ | :----------- | :--------------------------------------------- |
| 学生管理      | 增删改查     | 维护学生账号信息（学号、姓名、班级、专业等）   |
| 教师管理      | 增删改查     | 维护教师账号信息（工号、姓名、所属院系等）     |
| 班级/院系管理 | 基础数据维护 | 设置学校组织架构，便于后续按班级筛选统计       |
| 系统日志      | 操作记录查看 | 记录关键操作，便于追溯问题                     |
| 数据统计      | 全局概览     | 查看系统使用情况（用户数、考试场次、参与人次） |

### **1.2、教师端**

教师是系统的核心使用者，负责考试业务的完整闭环。根据市场调研，成熟的在线考试系统需要覆盖从题库建设→组卷→发布→监控→阅卷→分析的全流程

**（1）题库管理**

- 题目增删改查：支持单选题、多选题、判断题、填空题、简答题、编程题等基础题型
- 批量导入：支持Excel/Word模板批量导入题目，提升建库效率
- 题目分类：按科目、知识点、难度等级（易/中/难）进行标签化管理

**（2）试卷管理**

- 手动组卷：从题库中逐题挑选，自由组合试卷
- 自动组卷：设置题型数量、分值、知识点比例，系统随机抽题
- 试卷设置：设置考试时长、总分、及格线、开始/结束时间
- 试卷预览：组卷完成后以学生视角预览试卷效果

**（3）考试管理**

- 发布考试：选择试卷、指定参考班级/学生、设置考试时间
- 考试监控：实时查看考生状态（未开始/进行中/已交卷）、参考人数、异常情况
- 手动延长：为个别特殊考生延长考试时间

**（4）阅卷与成绩管理**

- 客观题自动阅卷：单选、多选、判断等客观题型，系统自动判分
- 成绩查询：查看考试最高分、最低分、平均分、及格率
- 成绩导出：将成绩表导出为Excel，便于线下存档
- 错题分析：查看每道题的正确率，分析学生薄弱知识点
- 学生管理：维护学生账号信息（学号，姓名，班级，专业等）

### **1.3、学生端**

| 功能模块 | 具体功能         | 设计说明                                                     |
| :------- | :--------------- | :----------------------------------------------------------- |
| 待考考试 | 查看待参加的考试 | 按时间倒序展示，显示考试名称、时长、起止时间                 |
| 在线考试 | 答题界面         | -顶部显示倒计时-左侧题号导航（已答/未答/当前题标识）-自动保存答案（防断电断网）-防切屏提醒（可配置切屏次数，超限自动交卷） |
| 考试记录 | 历史考试查询     | 查看已完成考试的成绩、答卷详情、正确答案                     |
| 错题本   | 自动收录错题     | 所有答错的题目自动归集，支持按科目筛选、反复练习             |
| 成绩单   | 个人成绩分析     | 展示各科成绩趋势图、知识点掌握雷达图                         |
| 个人信息 | 密码修改         | 学生可自行修改登录密码（账号由管理员统一管理）               |

## **2、项目亮点与进阶设计（可选）**

### **2.1、防作弊机制设计（提升项目含金量）**

- 防切屏监测：记录切屏次数，超过设定次数自动交卷
- 答题时间异常检测：交卷时间远短于正常用时，标记为可疑答卷
- 题目乱序/选项乱序：同一试卷不同考生看到题目顺序、选项顺序随机打乱

### **2.2、数据实时保存（用户体验提升）**

采用定时自动保存（如每30秒）或每答一题即保存，防止因断网断电导致答案丢失。

### **2.3、批量导入/导出功能（实用性体现）**

- 学生账号Excel批量导入
- 题目Excel批量导入（提供模板）
- 成绩Excel导出
